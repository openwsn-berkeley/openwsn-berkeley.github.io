This is a program which shows how to use the bsp modules for the board and leds.

Note
: Since the bsp modules for different platforms have the same declaration, you can use this project with any platform.

Load this program on your board. Open a serial terminal client (e.g. PuTTY or TeraTerm):

  • when you start the program, you should read "Hello World!" on your terminal client.
  • when you enter a character on the client, the board echoes it back (i.e. you see the character on the terminal client) and the "ERROR" led blinks.
