Kinetis Identify.

This is primarly a reporting function that displays information about the specific CPU to the default terminal including:

  • Kinetis family
  • package
  • die revision
  • P-flash size
  • Ram size
void flash_identify ( void  )

flash Identify

Returns
None

This is primarly a reporting function that displays information about the specific flash parameters and flash version ID for the current device. These parameters are obtained using a special flash command call "read resource." The first four bytes returned are the flash parameter revision, and the second four bytes are the flash version ID.
