Detailed Description

derfmega-specific definition of the "radiotimer" bsp module.

Z1-specific definition of the "radiotimer" bsp module.

LPCXpresso-specific definition of the "radiotimer" bsp module. this timer uses 2 match (compare) registers and 1 capture register. The time counter is a free counter. The match register 0 is used to control time slot size, it is re-scheduled after every match. The second match register is used to schedule events within the slot. The capture register is used to capture counter value at different moments, e.g the first bit of a packet leaves the radio.
